    Function Optional.dispatch

    Allows you to call dot operator on the internal value if present If there is no value inside, or it is null, dispatching will still work but will produce a series of no-ops.

    auto inout auto dispatch();

    If you try and call a manifest constant or static data on T then whether the manifest or static immutable data is called depends on if the instance it is called on is a some or a none.

    Returns

    A proxy that dispatched all dot operations to T if there is a T and returns an Optional of whatever T.blah would've returned. 

    struct A {
    struct Inner {
        int g() { return 7; }
    }
    Inner inner() { return Inner(); }
    int f() { return 4; }
}
auto a = some(A());
auto b = no!A;
auto b = no!(A*);
a.dispatch.inner.dispatch.g; // calls inner and calls g
b.dispatch.inner.dispatch.g; // no op.
b.dispatch.inner.dispatch.g; // no op.
